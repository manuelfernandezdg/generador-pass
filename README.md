# Generador de Contraseñas Seguras

Una aplicación web sencilla y ligera que genera contraseñas aleatorias y seguras.  
Funciona 100% en el navegador, **sin dependencias externas**, usando solo HTML, CSS y JavaScript puro (Vanilla JS).

## ✨ Características

- Elige la **longitud** de la contraseña (rango ajustable con slider y número)
- Opcional: incluir mayúsculas, números y símbolos
- Generación instantánea al enviar el formulario
- Botón para **copiar al portapapeles** con feedback visual
- Interfaz minimalista y responsive
- Todo el código es vanilla → sin frameworks ni librerías
